## Sobre o projeto

Esta landing page demonstra o <strong>Lumio.UI</strong>, um design system criado para facilitar o desenvolvimento de interfaces modernas, acessíveis e consistentes. O objetivo é fornecer uma base sólida de componentes reutilizáveis, estilos e boas práticas para acelerar a criação de produtos digitais.

## O que é o Lumio.UI?

Lumio.UI é um conjunto de componentes, tokens de design e diretrizes visuais que promovem padronização, escalabilidade e eficiência no desenvolvimento front-end. Ele foi pensado para ser flexível, personalizável e fácil de integrar em diferentes projetos.

## Principais características

- Componentes reutilizáveis e acessíveis
- Paleta de cores consistente
- Tipografia moderna
- Facilidade de customização
- Foco em usabilidade e experiência do usuário
